goby 介绍

goby 一个攻击面梳理工具、资产排查工具，用最少的包完成最多的数据整理，谁用谁说OK

开发说明

接口一共有三个模块 client、poc、asset 所有与客户端相关的都放到 client 与资产相关的放到 asset 与 poc 相关的放到 poc。通过抓包进行分析 api 得出接口
